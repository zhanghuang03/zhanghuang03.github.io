## Output plugin : Http

* Author: InterestingLab
* Homepage: https://interestinglab.github.io/waterdrop
* Version: 1.0.0

### Description

将Waterdrop处理的数据，通过调用Http接口输出，可以指定`Get/Post`方式，允许灵活指定url参数和request body.
