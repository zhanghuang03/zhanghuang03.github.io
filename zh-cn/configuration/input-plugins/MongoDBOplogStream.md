## Input plugin : MongoDBOplogStream [Streaming]

* Author: InterestingLab
* Homepage: https://interestinglab.github.io/waterdrop
* Version: 1.0.0

### Description

订阅MongoDB的`oplog.rs`集合的实时数据，获取Document的Insert/Update/Delete变更记录，在Waterdrop中分析和处理，最后输出。
