## Input plugin : MySQLBinlogStream [Streaming]

* Author: InterestingLab
* Homepage: https://interestinglab.github.io/waterdrop
* Version: 1.0.0

### Description

实时订阅MySQL的binlog变更记录，能够实现通过Waterdrop完成MySQL数据同步和ETL，支持CRUD。
